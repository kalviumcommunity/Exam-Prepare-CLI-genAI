# 📝 Exam Prepare CLI  
GenAI-powered command-line tool that creates personalized mock tests based on any topic and description you provide. It uses prompting, RAG (Retrieval-Augmented Generation), structured output, and function calling to simulate realistic exam preparation in seconds.

---

## ✨ Features

🧠 **Natural Prompting**  
Just describe a topic like “JavaScript closures” or “Cell structure in biology,” and the assistant will generate tailored multiple-choice questions.

🔍 **RAG (Retrieval-Augmented Generation)**  
Enhances question generation by retrieving supporting material from external documents or notes stored locally and integrated through vector search.

📊 **Structured Output**  
Returns mock test questions in a structured JSON format with clearly defined fields for question, options, and answer key.

⚙️ **Function Calling**  
Automatically saves generated tests to files using GenAI-powered function calls, so you can review or share them anytime.

---

## 🧠 How It Works

1. **Prompting** – You input a topic and description in the CLI  
2. **RAG** – The assistant searches a local knowledge base for supporting material  
3. **Structured Output** – Questions are returned in a consistent JSON format  
4. **Function Calling** – The tool saves your mock test automatically using AI function calls
